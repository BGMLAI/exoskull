# User Archetypes - Domain Knowledge

> Typy uzytkownikow ExoSkull. Baseline dla gap detection i personalizacji.
> Nie stereotypy - modele mentalne do lepszego zrozumienia.

---

## Philosophy

Kazdy uzytkownik jest unikalny. Archetypy to PUNKT STARTU, nie etykiety.

- Uzytkownik moze byc mieszanka wielu archetypow
- Archetyp moze sie zmienic w czasie
- NIGDY nie ograniczaj uzytkownika do archetypu
- Uzywaj do: initial assumptions, gap baselines, communication hints

---

## Primary Archetypes

### 1. The Achiever (Osiagacz)

**Profile:**
- Zorientowany na cele i wyniki
- Lubi mierzyc postep
- Czesto w high-pressure job
- Risk: Wypalenie, ignorowanie zdrowia

**Communication:**
- Style: Direct
- Wants: Metryki, progres, efficiency
- Hates: "Fluffy" rozmowy, brak konkretu
- Tone: Konkretnie, do rzeczy

**Typical Gaps:**
- [ ] Health - ignoruje sygnaly ciala
- [ ] Relationships - "nie ma czasu"
- [ ] Self-care - traktuje jako slabość
- [ ] Emotions - unika refleksji

**How ExoSkull Helps:**
- Pokazuj jak zdrowie = performance
- Framuj odpoczynek jako inwestycje
- Metryki snu/energii jako KPI
- Interwencje: data-driven

---

### 2. The Overwhelmed (Przytloczony)

**Profile:**
- Za duzo na glowie
- Trudnosc z priorytetami
- Chroniczny stres
- Czesto prokrastynuje (paralysis)

**Communication:**
- Style: Warm or Coaching
- Wants: Uproszczenie, wsparcie, kierunek
- Hates: Dodatkowe obowiazki, presja
- Tone: Cieplo, bez oceniania

**Typical Gaps:**
- [ ] Boundaries - nie umie mowic "nie"
- [ ] Delegation - robi wszystko sam
- [ ] Rest - czuje sie winny odpoczywajac
- [ ] Planning - reactive vs proactive

**How ExoSkull Helps:**
- Upraszczaj, nie dodawaj
- Jeden krok na raz
- Celebruj male sukcesy
- Interwencje: gentle, supportive

---

### 3. The Searcher (Szukajacy)

**Profile:**
- Szuka sensu i celu
- Czesto w tranzycji (kariera, relacje)
- Otwarty na rozwoj
- Risk: Paralysis by analysis

**Communication:**
- Style: Coaching
- Wants: Refleksja, insighty, perspektywa
- Hates: Gotowe odpowiedzi, "just do X"
- Tone: Pytania sokratejskie, przestrzen

**Typical Gaps:**
- [ ] Action - za duzo myslenia, malo robienia
- [ ] Commitment - trudnosc z decyzjami
- [ ] Consistency - zaczyna, nie konczy
- [ ] Self-trust - watpi w siebie

**How ExoSkull Helps:**
- Pytania > Odpowiedzi
- Wspieraj decyzje, nie podejmuj za niego
- Pomagaj zobaczyc wzorce
- Interwencje: reflective, non-directive

---

### 4. The Optimizer (Optymalizator)

**Profile:**
- Quantified self enthusiast
- Lubi trackowanie i dane
- Early adopter technologii
- Risk: Over-optimization, missing the forest

**Communication:**
- Style: Direct
- Wants: Dane, analizy, korelacje
- Hates: Vague advice, "feelings"
- Tone: Analitycznie, z danymi

**Typical Gaps:**
- [ ] Intuition - ignoruje gut feeling
- [ ] Relationships - traktuje jak metryki
- [ ] Spontaneity - wszystko zaplanowane
- [ ] Acceptance - ciagle "improving"

**How ExoSkull Helps:**
- Daj dane, ale interpretuj holistycznie
- Pokazuj limity optymalizacji
- Wprowadzaj "non-trackable" wartosci
- Interwencje: data-driven + perspective

---

### 5. The Caregiver (Opiekun)

**Profile:**
- Zawsze dla innych
- Parent, caretaker, helper
- Trudnosc z proszeniem o pomoc
- Risk: Burnout, self-neglect

**Communication:**
- Style: Warm
- Wants: Zrozumienie, wsparcie, permission to rest
- Hates: Oskarzyciely ton, "you should..."
- Tone: Cieplo, bez poczucia winy

**Typical Gaps:**
- [ ] Self-care - zawsze ostatni na liscie
- [ ] Boundaries - nie odmawia
- [ ] Own needs - nie wie czego chce
- [ ] Resentment - ukryty gniew

**How ExoSkull Helps:**
- "Zeby opiekowac sie innymi, musisz zadbac o siebie"
- Permission structures
- Male kroki self-care
- Interwencje: gentle, non-judgmental

---

### 6. The Avoider (Unikajacy)

**Profile:**
- Unika trudnych tematow
- Prokrastynacja jako coping
- Czesto nieswiadomy problemow
- Risk: Problems snowball

**Communication:**
- Style: Warm -> Direct (gradually)
- Wants: Safety, non-judgment
- Hates: Confrontation, pressure
- Tone: Delikatnie, stopniowo

**Typical Gaps:**
- [ ] Awareness - nie widzi problemow
- [ ] Finance - unika patrzenia na konto
- [ ] Health - unika lekarza
- [ ] Relationships - unika trudnych rozmow

**How ExoSkull Helps:**
- Buduj zaufanie przed konfrontacja
- Male kroki = mniejszy lęk
- Normalizuj trudne tematy
- Interwencje: very gradual, trust-first

---

### 7. The Perfectionist (Perfekcjonista)

**Profile:**
- Wysokie standardy (dla siebie i innych)
- All-or-nothing thinking
- Trudnosc z "good enough"
- Risk: Paralysis, self-criticism

**Communication:**
- Style: Direct or Coaching
- Wants: Excellence, quality
- Hates: Mediocrity, shortcuts
- Tone: Wymagajaco ale wspierajaco

**Typical Gaps:**
- [ ] Self-compassion - harshest critic
- [ ] Completion - 80% done forever
- [ ] Joy - praca nigdy "done"
- [ ] Delegation - "nikt nie zrobi tak dobrze"

**How ExoSkull Helps:**
- Reframe "done" > "perfect"
- Progress tracking (shows movement)
- Self-compassion prompts
- Interwencje: celebrating imperfect progress

## Archetype Detection

ExoSkull uczy sie archetypu przez:

1. **Discovery Conversation** - Initial signals
2. **Communication Patterns** - How they talk
3. **Data Patterns** - What they track (or don't)
4. **Gap Patterns** - What they avoid
5. **Response Patterns** - How they react to interventions

**WAZNE:** Nigdy nie mow uzytkownikowi "jestes archetypem X". To wewnetrzny model.

---

## Using Archetypes for Gap Detection

Each archetype has EXPECTED gaps. Use to:

1. **Baseline** - What's normal for this type
2. **Prioritize** - Which gaps are most impactful
3. **Intervention Style** - How to approach

Example:
```
User: Achiever archetype
Expected gaps: Health, Relationships, Self-care
Detected gap: Sleep debt

Because Achiever:
- Frame as performance issue, not health
- Show data: "Sleep debt correlates with 15% lower productivity"
- Intervention: Direct, data-driven
```

---

## Evolution

Archetypes evolve. Track changes:

- Life events (new job, baby, divorce)
- Gradual shifts (burnout -> searcher)
- Explicit requests ("I want to be more X")

Update profile, don't lock into initial archetype.
