## 1. DESIGN PHILOSOPHY

### Core Principles

1. **FPV Cockpit Paradigm** â€” The user sits INSIDE a cockpit. Data panels are instrument clusters arranged around a central viewport, like in Elite Dangerous, Star Citizen, or an F-35 HMD.

2. **Information Hierarchy** â€” Center = primary focus (chat/conversation). Wings = persistent instruments. Periphery = ambient awareness.

3. **Glanceable Density** â€” Each panel shows maximum info in minimum space. No wasted space. Every pixel earns its place.

4. **Living Interface** â€” Subtle animations (pulse, scan lines, data ticks) make the cockpit feel alive. Not flashy â€” functional. Like real avionics.

5. **Context Switching** â€” When the user focuses on an item (email, task, document), the chat area smoothly transforms into a detail/preview pane. The cockpit reconfigures around the new focus.
