# CZĘŚĆ II: TEORIA PĘTLI NIEWYSTARCZALNOŚCI

## 2.1 Definicja

**Pętla niewystarczalności** to samonapędzający się mechanizm, w którym:
1. System norm definiuje co znaczy być "wystarczającym"
2. Jednostka internalizuje te normy jako wewnętrzny głos
3. Niespełnienie norm → poczucie niewystarczalności → kompensacja
4. Kompensacja wzmacnia strukturę norm
5. Cykl się powtarza

## 2.2 Pętla męska (wartość przez OSIĄGNIĘCIA)

**Precarious Manhood Theory (Vandello & Bosson 2008):**
- Męskość jest statusem do ZDOBYCIA (nie wrodzona)
- Może być UTRACONA przez porażkę
- Wymaga CIĄGŁEJ demonstracji

**Empiryczne potwierdzenie:**
- 5 eksperymentów, d ≈ 0.4-0.6
- Badanie 62 krajów (N=33,417) — PMB istnieje globalnie
- Korelacja negatywna z HDI i równością płci

**Normy męskości (spirala 6 poziomów):**
```
0. DZIECIŃSTWO (0-6)    → "Chłopcy nie płaczą"
1. SZKOŁA (6-12)        → Wzmocnienie przez nauczycielki
2. ADOLESCENCJA (12-18) → "Ciota", "mięczak"
3. PARTNERSTWO (18-35)  → Słabość = utrata atrakcyjności
4. KRYZYS (30-50)       → Rozwód, utrata dzieci
5. SKUTKI               → Śmierć lub chroniczne cierpienie
```

**Pułapka bez wyjścia:**

| Ścieżka A: Spełniaj normy | Ścieżka B: Odrzuć normy |
|---------------------------|-------------------------|
| Zarabiaj dużo | Okazuj emocje |
| Bądź silny | Proś o pomoc |
| Nie okazuj słabości | Nie rywalizuj |
| **Koszt:** wypalenie, śmierć | **Koszt:** utrata statusu, odrzucenie |

## 2.3 Pętla kobieca (wartość przez PRZYNALEŻNOŚĆ)

**Konkurencja międzykobieca (Vaillancourt 2013):**
- Indirect aggression: plotki, wykluczenie, niszczenie reputacji
- Skierowana głównie przeciw atrakcyjnym kobietom
- Slut-shaming jako strategia konkurencyjna

**Hypergamia + rosnący status kobiet:**
- Preferencja partnera o równym/wyższym statusie
- Kobiety podnoszą swój status (edukacja, praca)
- Pool "akceptowalnych" mężczyzn maleje
- "Gdzie są wszyscy dobrzy mężczyźni?"

**Dane z aplikacji randkowych:**
- Kobiety oceniają ~80% mężczyzn jako "poniżej średniej"
- Mężczyźni rozkładają oceny wzdłuż krzywej normalnej

## 2.4 Sprzężenie pętli

```
PĘTLA MĘSKA                    PĘTLA KOBIECA
     |                              |
     v                              v
Niewielu "wygranych"          Konkurencja o "top" mężczyzn
     |                              |
     v                              v
Wielu "przegranych"           Frustration + podnoszenie poprzeczki
     |                              |
     +-----> ESKALACJA <------------+
```

**Monogamia jako stabilizator (Henrich 2012):**
- Każdy mężczyzna ma szansę na partnerkę
- "Top" mężczyźni nie monopolizują kobiet
- Redukcja intensywności konkurencji

# CZĘŚĆ III: TRANSMISJA NORM

## 3.1 Asymetria ekspozycji czasowej

| Środowisko | % kobiet |
|------------|----------|
| Przedszkole | 98% |
| Szkoła podstawowa | 87% |
| Rodzina (niepełna) | 70%+ dzieci przy matkach |

**Implikacja:** Chłopiec uczy się "jak być mężczyzną" głównie od KOBIET.

## 3.2 Mechanizmy transmisji

**Matki:**
- Silniejszy implicit bias przeciw emocjonalności synów (IAT)
- Mniej słów emocjonalnych z synami niż córkami
- Transmitują normę "man up" ze strachu przed oceną społeczną syna
- Nieco więcej surowej dyscypliny wobec synów (d = 0.12)

**Ojcowie:**
- Egzekwują normy męskości bardziej jawnie i rygorystycznie
- Silniejsze jawne przekonania stereotypowe (d = 0.38)
- Mniej akceptujący wobec niekonformizmu płciowego synów
- ALE: mniej dostępni emocjonalnie (tylko 4-7% synów szuka u nich wsparcia)

**Nauczycielki:**
- Oceniają chłopców surowiej za identyczne prace (d = 0.30 w matematyce)
- Dyscyplinują chłopców częściej i surowiej
- Bias nie wynika z płci nauczyciela — struktury szkolne faworyzują "kobiece" cechy

**Rówieśnicy:**
- Główni egzekutorzy norm od adolescencji
- Epitety homofobiczne i fizyczne policjonowanie
- Niekonformizm płciowy karany surowiej u chłopców niż dziewczynek

## 3.3 Konsekwencje — aleksytymia

**Aleksytymia** (trudność w identyfikowaniu/wyrażaniu emocji):
- Dotyka mężczyzn 2x częściej niż kobiety
- Hipoteza Normative Male Alexithymia (Levant)
- Chłopcy zaczynają życie bardziej ekspresyjni, do 4 roku stają się mniej

**Łańcuch przyczynowy:**
```
Socjalizacja → Tłumienie emocji → Aleksytymia → 
→ Trudności w regulacji → Eksternalizacja (przemoc, SUD) → 
→ Problemy zdrowotne → Przedwczesna śmierć
```

# CZĘŚĆ IV: TRIADA KONTROLI

## 4.1 Trzy operacje patologicznej kontroli

**1. ZNIEKSZTAŁCENIE MAPY**
- "Twoja percepcja jest błędna"
- Gaslighting, kłamstwa, "przesadzasz"
- Ja definiuję co jest prawdą

**2. KONTROLA PRZEPŁYWU**
- "Twój dostęp do zasobów zależy ode mnie"
- Izolacja od rodziny, kontrola pieniędzy
- Blokowanie kontaktu z dziećmi

**3. ODWRÓCENIE WEKTORA**
- "To twoja wina"
- DARVO (Deny, Attack, Reverse Victim and Offender)
- Projekcja, "patrz jak się zachowujesz"

## 4.2 Triada jako struktura ZDROWEJ komunikacji

| Element | Patologia | Zdrowie |
|---------|-----------|---------|
| MAPA | Zniekształcona | Wspólna rzeczywistość |
| PRZEPŁYW | Zablokowany | Otwarty dostęp |
| WEKTOR | Odwrócony | Jasna odpowiedzialność |

# CZĘŚĆ V: KONTRAKT PŁCI

## 5.1 Struktura historycznego kontraktu

**Mężczyzna dawał:**
- Ochrona (włącznie z gotowością do śmierci)
- Zasoby (utrzymanie rodziny)
- Status społeczny (kobieta zyskiwała przez męża)
- Ryzyko (wojna, niebezpieczna praca)

**Mężczyzna dostawał:**
- Pewność ojcostwa (wierność seksualna)
- Władza decyzyjna
- Lojalność partnerki
- Szacunek społeczny
- Dostęp seksualny

## 5.2 Rozpad kontraktu

**Jednostronność rozpadu:**
- Kobiety wyszły z zobowiązań (emancypacja — słusznie)
- Kobiety zachowały korzyści (ochrona, alimenty)
- Kobiety zyskały nowe prawa (głosowanie, praca, własność)
- Mężczyźni zostali z obowiązkami (zarabianie, ochrona)
- Mężczyźni stracili korzyści (lojalność, pewność)
- Mężczyźni NIE zyskali nowych praw

**Biasy blokujące renegocjację:**
- Women Are Wonderful effect
- In-group bias kobiet (silny) vs mężczyzn (brak/outgroup)
- Rycerskość (mężczyźni chronią kobiety kosztem siebie)
- Gamma bias (wyolbrzymianie męskiej sprawczości + kobiecego cierpienia)

- Negatywny self-talk → wyższy kortyzol, ciśnienie, RPE

- Wzrost sprawczości narracyjnej POPRZEDZA poprawę w terapii

**Tłumienie emocji → śmiertelność (Harvard, 12 lat):**
- 75. vs 25. percentyl tłumienia
- HR = 1.35 dla śmiertelności ogólnej
- HR = 1.70 dla śmiertelności nowotworowej

# CZĘŚĆ VII: ONTOLOGIA PRZEFAZOWYWANIA

## 7.1 Centralna teza

**ŻYCIE = PRZEFAZOWYWANIE NA TLE**

## 7.2 Elementy

| Symbol | Nazwa | Znaczenie |
|--------|-------|-----------|
| A | tło | możliwość, pole |
| B | akt | realizacja, zdarzenie |
| τ = A-B | napięcie | różnica potencjałów |
| Φ | przefazowywanie | dynamika A↔B |

## 7.3 Równania

**Równanie przefazowywania:**
```
Φ = τ · A = (A - B) · A
```

**Równanie ewolucji:**
```
dA'/dt = (A - B) · A
```

**Warunki brzegowe:**
- Φ → 0 gdy τ → 0 lub A → 0 (śmierć)
- Ω > 0 ⟺ życie

## 7.4 Operatory

| Operator | Efekt | Interpretacja |
|----------|-------|---------------|
| AA | A ⊗ A | trwanie |
| AB | A ⊗ B | zmiana |
| AA · AB | pełna struktura | życie które trwa i rośnie |

## 7.5 Spirala jako rozwiązanie

**Koło (Wschód):** A → B → A (powrót do identycznego)
**Linia (Zachód):** A → A' → A'' (postęp bez powrotu)
**Spirala:** A → B → A' → B' → A'' (powrót + postęp)

```
Spirala jest JEDYNYM kształtem który:
1. Wraca (jak koło)
2. Ale nigdy do tego samego miejsca (jak linia)
```

```
ZNANE <---------- napięcie ----------> NIEZNANE

```

| Poziom | Znane (ja) | Napięcie | Nieznane (nie-ja) |

      (zablokowane, wraca do tego samego ja)

# CZĘŚĆ IX: ZASTOSOWANIE

## 9.1 Dla zrozumienia

Śmiertelność mężczyzn to skutek systemowej blokady kompensacji — napięcie które nie może się rozwiązać, przepływ który jest zablokowany.

## 9.2 Dla terapii

- Nie naprawiaj "złych zachowań" (wentyle)
- Odblokuj przepływ
- Przywróć możliwość przefazowywania
- Pozwól by A stało się A', nie wracało do A

## 9.3 Dla relacji

**Zdrowa relacja = zdrowa Triada:**
- Wspólna mapa
- Otwarty przepływ
- Jasny wektor

## 9.4 Dla jednostki

> Wyjście z pętli to nie wybór między opcją A i B. 
> To zmiana TŁA — przejście z "moja wartość zależy od X" do "jestem wartościowy, kropka."
