# KOMBINATORYKA PRZEFAZOWYWANIA
## Pełna mapa form organizacji A i B

---

## 1. KOMBINACJE BINARNE

### Macierz 2×2

|        | **A (tło)**  | **B (akt)**   |
|--------|--------------|---------------|
| **A**  | AA (trwanie) | AB (życie)    |
| **B**  | BA (szukanie)| BB (rozpad)   |

---

### AA — Tło × Tło

**Formuła:** A ⊗ A  
**Znaczenie:** Głębokie pole możliwości bez realizacji  
**Napięcie:** τ = 0  

| Aspekt | Opis |
|--------|------|
| **Krótkoterminowo** | Stabilność, zachowanie formy, akumulacja potencjału |
| **Długoterminowo** | Stagnacja, brak rozwoju, entropia wewnętrzna |
| **Patologia** | Skostnienie, depresja, śmierć przez nieużywanie |

**Przykłady:**
- Biologia: DNA bez ekspresji — genom nieaktywny
- Psyche: Nieświadomość bez aktywacji — cień ukryty
- Fizyka: Funkcja falowa bez pomiaru — superpozycja
- Ekonomia: Kapitał bez inwestycji — martwy pieniądz

**Bateson:** Pętla bez wejścia zewnętrznego — system zamknięty  
**Jung:** Persona bez cienia — fałszywa integralność  
**Archetyp:** Mędrzec zamknięty w wieży

---

### AB — Tło × Akt

**Formuła:** A ⊗ B  
**Znaczenie:** Możliwość realizująca się. Pole z napięciem = ruch = życie.  
**Napięcie:** τ = max  

| Aspekt | Opis |
|--------|------|
| **Krótkoterminowo** | Ruch, realizacja, manifestacja |
| **Długoterminowo** | Rozwój, wzrost, wzbogacenie tła |
| **Patologia** | Brak (to jest zdrowa forma) |

**Przykłady:**
- Biologia: DNA → transkrypcja → białko
- Serce: Rozkurcz → skurcz
- Fizyka: Fala → pomiar → cząstka
- Psyche: Potencjał → działanie → doświadczenie

**Bateson:** Różnica która robi różnicę — informacja  
**Jung:** Indywiduacja — integracja świadomości z nieświadomością  
**Archetyp:** Bohater wyruszający z domu

---

### BA — Akt × Tło

**Formuła:** B ⊗ A  
**Znaczenie:** Realizacja szukająca oparcia. Działanie bez fundamentu.  
**Napięcie:** τ ≈ 0.7 (niestabilne)  

| Aspekt | Opis |
|--------|------|
| **Krótkoterminowo** | Poszukiwanie, eksploracja, próba zakorzenienia |
| **Długoterminowo** | Albo znajdzie tło (→AB), albo rozpłynie się (→BB) |
| **Patologia** | Niepokój, lęk egzystencjalny, bezdomność |

**Przykłady:**
- Biologia: Białko szukające receptora
- Psyche: Impuls szukający sensu — acting out
- Społeczeństwo: Rewolucja szukająca tradycji — legitymizacja
- Ekonomia: Spekulacja szukająca wartości realnej

**Bateson:** Komunikat bez kontekstu — potencjalna schizofrenia  
**Jung:** Cień szukający integracji  
**Archetyp:** Błędny rycerz

---

### BB — Akt × Akt

**Formuła:** B ⊗ B  
**Znaczenie:** Czyste działanie bez tła. Ruch który się rozprasza.  
**Napięcie:** τ = 0 (brak różnicy — ale przez brak A)  

| Aspekt | Opis |
|--------|------|
| **Krótkoterminowo** | Rozproszenie, entropia, utrata formy |
| **Długoterminowo** | Anihilacja, śmierć przez wyczerpanie |
| **Patologia** | Rozpad, psychoza, destrukcja |

**Przykłady:**
- Biologia: Komórka rakowa — niekontrolowana proliferacja
- Psyche: Mania — działanie bez refleksji
- Fizyka: Foton — czysta energia, brak masy spoczynkowej
- Społeczeństwo: Rewolucja permanentna

**Bateson:** Schizmogeneza symetryczna — eskalacja bez hamulca  
**Jung:** Inflacja ego — utożsamienie z archetypem  
**Archetyp:** Trickster bez granic

## 2. KOMBINACJE TERNARNE

### Spektrum od życia do śmierci

| Kombinacja | Nazwa | Stan | Przykład |
|------------|-------|------|----------|
| **AAA** | Tło³ | Śmierć przez niebycie | Wszechświat przed Big Bangiem |
| **AAB** | Głębokie tło + akt | Życie dojrzałe | Mistrz — głęboka wiedza w precyzyjnym działaniu |
| **ABA** | Tło-akt-tło | Spirala wznosząca | Oddech — pełny cykl |
| **ABB** | Zakorzenione podwójne działanie | Flow state | Sportowiec w szczycie formy |
| **BAA** | Akt szukający głębokiego tła | Konwersja | Rewolucjonista budujący instytucje |
| **BAB** | Akt-tło-akt | Niestabilna równowaga | Jonglowanie |
| **BBA** | Podwójny akt + tło | Wypalenie → odpoczynek | Kryzys prowadzący do transformacji |
| **BBB** | Akt³ | Anihilacja | Eksplozja — czysta energia bez formy |

## 3. DYNAMIKA PRZEJŚĆ

### Przejścia zdrowe (→)

```
AA ──τ↑──→ AB    Ożywienie (napięcie rośnie)
AB ──τ↓──→ AA    Odpoczynek (napięcie maleje)
BA ──A!──→ AB    Zakorzenienie (znalezienie tła)
```

### Przejścia patologiczne (⤳)

```
AB ──A↓──⤳ BB    Rozproszenie (utrata tła)
BA ──A×──⤳ BB    Dalszy rozpad (porażka szukania)
AA ──!──⤳ BB     Eksplozja (nagły wybuch bez przejścia)
```

### Przejścia kompensacyjne (⇢)

```
BB ──A?──⇢ BA    Desperacja (szukanie tła)
BB ──X──⇢ AA     Kolaps (reset/śmierć)
```

## 4. HIERARCHIA FORM ŻYCIA

### Optymalny stan

```
╔════════════════════════════════════════╗
║  AA × AB = Ω  (życie które trwa i rośnie)  ║
║  Spirala wznosząca                      ║
╚════════════════════════════════════════╝
```

### Hierarchia (od najzdrowszej)

1. **AA × AB** — Optimum: trwanie wspiera zmianę
2. **AB** — Aktywne życie, ale ryzyko wypalenia
3. **AA** — Życie uśpione, potencjał niewykorzystany
4. **BA** — Stan przejściowy, poszukiwanie
5. **BB** — Rozpad, entropia

### Stany graniczne

```
AAA → 0     Śmierć przez niebycie
BBB → ∞     Śmierć przez rozproszenie
```

## 5. ZASTOSOWANIA KLINICZNE

### Psychopatologia jako dysfunkcja kombinacji

| Stan | Kombinacja | Mechanizm |
|------|------------|-----------|
| Depresja | AA (nadmiar) | Brak napięcia, wycofanie z działania |
| Mania | BB (nadmiar) | Działanie bez zakorzenienia |
| Lęk | BA (dominacja) | Działanie bez oparcia |
| Zdrowie | AA × AB | Równowaga trwania i zmiany |

### Terapia jako przywracanie równowagi

```
Depresja (AA) → wprowadzić napięcie → AB
Mania (BB) → szukać tła → BA → AB
Lęk (BA) → zakorzenić → AB
```

## 6. FORMALIZACJA

### Operatory

```
AA = A ⊗ A     iloczyn tensorowy tła z sobą
AB = A ⊗ B     iloczyn tensorowy tła z aktem
BA = B ⊗ A     iloczyn tensorowy aktu z tłem
BB = B ⊗ B     iloczyn tensorowy aktu z sobą
```

### Uwaga o nieprzemienności

**AB ≠ BA**

- AB: tło jest fundamentem, akt jest realizacją na tym tle
- BA: akt jest pierwszy, szuka tła retrospektywnie

To nie jest mnożenie liczb — kolejność ma znaczenie ontologiczne.

### Warunki życia

```
Ω > 0  ⟺  AA · AB > 0  ⟺  A² · B > 0

Gdzie:
- A > 0 (musi być tło)
- B > 0 (musi być akt)
- A² > B (tło musi dominować nad aktem)
```

---

## 7. WNIOSKI

1. **Życie wymaga obu** — ani samo AA, ani samo BB nie podtrzymuje życia
2. **Kolejność ma znaczenie** — AB ≠ BA (zakorzenione działanie vs. działanie szukające korzeni)
3. **Optimum to iloczyn** — AA × AB = trwanie × zmiana
4. **Patologie to ekstrema** — AAA (niebycie) i BBB (rozproszenie) to dwie formy śmierci
5. **Terapia to równoważenie** — przywracanie relacji AA:AB
