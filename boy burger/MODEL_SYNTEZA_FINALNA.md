## PĘTLA NIEWYSTARCZALNOŚCI — DEFINICJA

```
PĘTLA NIEWYSTARCZALNOŚCI =

    Powtarzająca się sekwencja
    niemożności realizacji
    wzrostu harmonijnej złożoności
    przez niedopasowanie
    ZNANEGO (postrzegane zasoby)
    do
    NIEZNANEGO (postrzegane tło)
```

### Mechanizm pętli:

```
Iteracja → uczenie (Δ) → ale Δ < próg
Iteracja → uczenie (Δ) → ale Δ < próg
...
Σ(Δ) nigdy nie osiąga masy krytycznej

System "uczy się" ale nie wystarcza
by wywołać przefazowanie transformujące
```

### Próg = odległość od równowagi (0)

```
Im dalej od 0 → tym wyższy próg → tym trudniej przefazować
Im bliżej 0 → tym niższy próg → tym łatwiej przefazować

PARADOKS: Pętla PRODUKUJE napięcie → próg rośnie → trudniej wyjść
          PĘTLA SIĘ SAMOWZMACNIA
```

## ŹRÓDŁA NIEDOPASOWANIA

Trzy punkty gdzie informacja może być zaburzona:

| PUNKT | ZABURZENIE | EFEKT |
|-------|------------|-------|
| Znane (zasoby) | Fałszywy obraz tego co mam | "Nie potrafię", "Nie mam" |
| Nieznane (tło) | Fałszywy obraz tego co przede mną | "Świat jest wrogi", "Nie ma wyjścia" |
| Relacja | Fałszywy obraz dopasowania | "To nie dla mnie", "Nie zasługuję" |

### Manipulacja = obniżanie postrzeganej wartości

- Obniżenie obrazu zasobów
- Obniżenie obrazu tła (możliwości w nim)
- Obniżenie obrazu relacji (siebie w relacji do świata)

### Słowo/informacja = instalator filtrów

Słowo działa jak instalacja która ustawia jak postrzegasz znane i nieznane.

Dzieciństwo: instalacja bazowych filtrów
Aktualna manipulacja: bieżące zaburzanie obrazu

```

    0 ←→ nie-0

POZIOM 1:

    ZASOBY ←→ TŁO

    Przefazowanie (wybór: znane → nieznane)
    Cykl: decyzja → ruch → nowe tło → decyzja...

POZIOM 5:
    Pętla niewystarczalności
    = chroniczne niedopasowanie znanego do nieznanego
    = wybory które nie budują adekwatnej złożoności
```

### Unikalne wkłady modelu:

- Struktura ZNANE ↔ NIEZNANE jako fundament świadomości
- Adekwatność (nie maksymalizacja) jako kryterium zdrowia
- Manipulacja jako obniżanie postrzeganej wartości
- Próg = odległość od równowagi (samowzmacnianie pętli)

## IMPLIKACJE DLA TERAPII (TPN)

### Diagnoza:
Gdzie jest niedopasowanie?
- Fałszywy obraz zasobów (znanego)?
- Fałszywy obraz tła (nieznanego)?
- Fałszywy obraz relacji między nimi?

### Interwencja:
Przywracanie adekwatnego postrzegania:
- Kalibracja obrazu zasobów (co faktycznie mam/potrafię)
- Kalibracja obrazu tła (co faktycznie jest możliwe)
- Rekonstrukcja relacji (jak moje znane pasuje do nieznanego)

### Cel:
Nie: maksymalny wzrost
Nie: brak napięcia
Ale: **złożoność adekwatna do tła** — znane dopasowane do nieznanego

### Mechanizm zmiany:
Dostarczenie energii do przekroczenia progu:
- Zwiększ Δ na iterację (intensywna interwencja)
- Zmniejsz rozpraszanie (utrzymanie świadomości)
- Obniż próg (zmiana kontekstu, usunięcie manipulatora)
