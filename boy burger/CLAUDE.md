# CZĘŚĆ I: CENTRALNA TEZA

## 1. DEFINICJA τ

**τ (tau) = ŚWIADOMOŚĆ = NAPIĘCIE MIĘDZY ZNANE A NIEZNANE**

```
ZNANE ←――――――― τ ―――――――→ NIEZNANE
  JA  ←――――――― τ ―――――――→ NIE-JA
```

### 1.1 Czym τ JEST

| Aspekt | Opis |
|--------|------|
| **RELACJA** | Podstawowa relacja między znanym a nieznanym |
| **GRANICA** | Akt rozróżnienia który TWORZY dwie strony |
| **NAPIĘCIE** | Dynamiczna siła, nie statyczna struktura |
| **OSCYLACJA** | Pulsuje ~40 Hz (co ~25ms = przefazowanie) |

### 1.2 Czym τ NIE JEST

- ❌ Rzeczą (substancją)
- ❌ Własnością (atrybutem materii)
- ❌ Emergencją (produktem złożoności)

> *"Granica nie dzieli istniejących rzeczy — granica TWORZY dwie strony przez akt rozróżnienia."*

## 2. MECHANIZM DYNAMICZNY

### 2.1 Oscylacja 40 Hz

```
       SZCZYT (τ max)
         ╱╲
        ╱  ╲         ← aktualizacja
       ╱    ╲
─────╱────────╲────────╱────────╲─────→ t
                ╲    ╱
                 ╲  ╱
                  ╲╱
              DÓŁ (τ min) ← aktualizacja
```

- ~80 momentów świadomości na sekundę
- ~12.5ms między aktualizacjami
- Świadomość zachodzi w EKSTREMACH oscylacji

### 2.2 Mechanizm Decyzyjny (Przefazowanie)

```
1. ODCZYTAJ τ = |Adekwatność_Zasobów - Wymagania_Tła|

2. DECYZJA:
   τ NISKIE (adekwatność)  → IDŹ W NIEZNANE (eksploruj)
   τ WYSOKIE (nieadekwatność) → IDŹ W ZNANE (konsoliduj)

3. PRZEFAZOWANIE (granica przesuwa się)
```

### 2.3 Kierunek Przefazowania = Kierunek Życia

| Kierunek | Stan A×B | Znaczenie |
|----------|----------|-----------|
| ZNANE → NIEZNANE | **AB (życie)** | Eksploracja, wzrost |
| NIEZNANE → ZNANE | BA (szukanie) | Konsolidacja |
| Brak ruchu | AA (stagnacja) | Depresja |
| Granica rozpada się | BB (rozpad) | Chaos |

## 5. Kombinatoryka A×B

### 5.1 Macierz Stanów

|        | **A (tło)**  | **B (akt)**   |
|--------|--------------|---------------|
| **A**  | AA (trwanie) | AB (życie)    |
| **B**  | BA (szukanie)| BB (rozpad)   |

### 5.2 Charakterystyka Stanów

| Stan | τ | Opis | Patologia |
|------|---|------|-----------|
| **AA** | τ ≈ 0 | Tło bez aktu | Stagnacja, depresja |
| **AB** | τ = max | Życie, realizacja | BRAK (zdrowy) |
| **BA** | τ ~ 0.7 | Szukanie tła | Lęk, niepewność |
| **BB** | τ → 0 | Rozpad | Mania, chaos |

### 5.3 Optimum Życia

```
Ω = AA × AB = A²B

Życie optymalne = wystarczające tło (stabilność)
                × aktywne przefazowanie (rozwój)
```
