## 55.21.1 Kryteria Dobrej Predykcji

- **Mierzalna** — operacjonalizowalna

## 55.21.2 Predykcje Neurobiologiczne

### P1: Oscylacja 40Hz jako korelat τ

**Predykcja:**
```
Subiektywna intensywność świadomości koreluje z AMPLITUDĄ
oscylacji gamma (30-100Hz, peak ~40Hz), a NIE tylko z obecnością.
```

**Test:**
- Mierzyć EEG podczas zadań o różnym poziomie świadomości
- Korelować amplitudę gamma z samoopisem intensywności doświadczenia

**Istniejące dane wspierające:**
- Dehaene & Changeux (2011): gamma synchrony w świadomym przetwarzaniu
- Engel & Singer (2001): binding hypothesis

**Unikalne dla τ:**
Inne teorie (GWT, IIT) mówią o obecności/nieobecności świadomości. τ mówi o INTENSYWNOŚCI jako funkcji amplitudy oscylacji.

---

### P2: Przefazowanie = spike aktywności

**Predykcja:**
```
W momencie "aha!" (insight) lub decyzji:
1. Gamma spike (>2 SD powyżej baseline)
2. Następuje 12.5-25ms PO spike'u subiektywne "wyładowanie"
3. Zmiana w default mode network connectivity
```

**Test:**
- fMRI + EEG podczas zadań insight (RAT, CRA)
- Mierzyć gamma spikes i subiektywne raporty

**Istniejące dane wspierające:**
- Jung-Beeman (2004): gamma burst przy insight
- Kounios & Beeman (2014): neural dynamics of insight

---

### P3: Depresja = obniżona zmienność gamma

**Predykcja:**
```
Osoby z depresją (vs kontrola):
1. Niższa amplituda gamma
2. Niższa ZMIENNOŚĆ gamma (mniejsze wahania)
3. Niższa connectivity między prefrontal a limbic

Interpretacja τ: "zablokowane przefazowanie" = mniejsza oscylacja
```

**Test:**
- EEG comparison: depresja vs kontrola
- Korelacja z skalami depresji (BDI, PHQ-9)

**Istniejące dane wspierające:**
- Fitzgerald & Watson (2018): reduced gamma in depression
- Liston et al. (2014): connectivity changes in MDD

## 55.21.3 Predykcje Psychologiczne

### P4: Terapia skuteczna = przywrócenie oscylacji τ

**Predykcja:**
```
Skuteczność terapii (CBT, EMDR, mindfulness) koreluje z:
1. Wzrostem HRV (heart rate variability)
2. Wzrostem gamma amplitude
3. Wzrostem eksploracyjnych zachowań

HRV jest proxy dla "zdolności do oscylacji" (vagal tone)
```

**Test:**
- Longitudinal: mierzyć HRV przed/po terapii
- Korelować zmianę HRV ze zmianą symptomów

**Istniejące dane wspierające:**
- Porges (2011): polyvagal theory
- Kemp et al. (2010): HRV and depression

---

### P5: Trauma = zamrożenie τ na specyficznej granicy

**Predykcja:**
```
PTSD charakteryzuje się:
1. Normalna ogólna gamma amplitude
2. ALE: obniżona gamma przy bodźcach związanych z traumą
3. Zamrożona connectivity między amygdala a prefrontal przy tych bodźcach

Interpretacja τ: granica "zamrożona" w miejscu traumy
```

**Test:**
- EEG/fMRI podczas ekspozycji na neutralne vs trauma-related cues
- Porównanie PTSD vs kontrola

**Istniejące dane wspierające:**
- Lanius et al. (2010): dissociative subtype PTSD
- van der Kolk (2014): body keeps the score

---

### P6: Flow = optimum τ (α ≈ 1)

**Predykcja:**
```
Stan flow (Csikszentmihalyi) charakteryzuje się:
1. Umiarkowane τ (nie za niskie, nie za wysokie)
2. Wysoka gamma coherence między areas
3. α ≈ 1 (challenge = skill)
4. Optymalny HRV
```

**Test:**
- Indukować flow (gaming, music, sport)
- Mierzyć EEG, HRV, samoraport

**Istniejące dane wspierające:**
- Csikszentmihalyi (1990): flow theory
- Katahira et al. (2018): neural correlates of flow

## 55.21.4 Predykcje Społeczne

### P7: Manipulacja = obniżanie τ partnera

**Predykcja:**
```
W relacjach z coercive control:
1. Ofiara ma chronically obniżone HRV
2. Ofiara ma obniżoną gamma variability
3. Ofiara wykazuje obniżoną eksplorację (mniejsze V)
4. Różnica HRV: kontroler > ofiara

Interpretacja τ: kontroler obniża adekwatność ofiary
```

**Test:**
- Porównanie par z/bez coercive control
- Mierzyć HRV, behawioralne proxy eksploracji

**Istniejące dane wspierające:**
- Stark (2007): coercive control theory
- Johnson (2008): typology of IPV

---

### P8: Polaryzacja = redukcja τ przez echo chambers

**Predykcja:**
```
Echo chambers (social media):
1. Redukują ekspozycję na NIEZNANE (B maleje)
2. Redukują τ (napięcie)
3. Prowadzą do sztywności poznawczej
4. Obniżają tolerance for ambiguity

Mierzalne: osoby w echo chambers mają niższą tolerance for ambiguity
```

**Test:**
- Mierzyć "echo chamber score" (diversity of media diet)
- Korelować z Tolerance for Ambiguity Scale (Budner)

## 55.21.5 Predykcje Fizyczne/Systemowe

### P9: Systemy na "edge of chaos" = optymalne τ

**Predykcja:**
```
Systemy adaptacyjne (organizmy, ekosystemy, organizacje):
1. Optymalne funkcjonowanie na "edge of chaos" (Kauffman)
2. Edge of chaos = α ≈ 1 w naszej terminologii
3. Mierzalne przez: 1/f noise, power-law distributions

Systemy za daleko od edge = nieadaptacyjne
```

**Test:**
- Mierzyć "criticality" systemów biologicznych
- Korelować z adaptacyjnością/zdrowiem

**Istniejące dane wspierające:**
- Kauffman (1993): edge of chaos
- Bak (1996): self-organized criticality

---

### P10: Entropia a τ

**Predykcja:**
```
W systemach termodynamicznych:
1. Gradient entropii ∝ τ
2. Życie utrzymuje stały gradient (Prigogine)
3. Śmierć = gradient → 0

Mierzalne: entropia produkcji w żywych vs martwych systemach
```

**Test:**
- Mierzyć entropy production rate w żywych organizmach
- Porównać z systemami nieożywionymi

**Istniejące dane wspierające:**
- Prigogine (1977): dissipative structures
- England (2013): entropy and life

## 55.21.6 Unikalne Predykcje τ (których inne teorie NIE robią)

### P11: Kierunek przefazowania ma znaczenie

**Predykcja:**
```
NIEZNANE → ZNANE (integracja) vs ZNANE → NIEZNANE (eksploracja)
mają RÓŻNE neuronalne korelaty:

- Integracja: activation w default mode network
- Eksploracja: activation w salience network

IIT/FEP/GWT nie robią tego rozróżnienia.
```

**Test:**
- fMRI podczas zadań integracyjnych vs eksploracyjnych
- Contrast DMN vs SN activation

---

### P12: Świadomość jest TEMPO, nie obecność

**Predykcja:**
```
Różnica między:
- Głęboki sen vs REM vs czuwanie
NIE jest binarna, ale GRADIENTOWA.

Mierzona jako: częstotliwość przefazowań (Hz)
- Głęboki sen: <1 Hz
- REM: ~10 Hz
- Czuwanie: ~40 Hz
- Medytacja deep: zmienne (2-100 Hz)
```

**Test:**
- Mierzyć gamma frequency podczas różnych stanów świadomości
- Korelować z subiektywnym "poziomem" świadomości
